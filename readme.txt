Limit WordPress User Registration to Invited Users with User Profiles Made Easy.

== Description ==
[Invitation Codes for User Profiles Made Easy](http://profileplugin.com/invitation-codes-addon/) is a free addon provided for limiting the registration only to invited users. Admin can enable/disable invitation code based registration anytime. Once enabled, users without valid invitation code can’t create a new registration.

Please note that this is an addon for  [User Profiles Made Easy](https://goo.gl/Dtqxsx) plugin and you need at least UPME version 2.1.10 to use it.

> **Main Features**
>
> * Create and manage invitation codes and quota
> * Send invitations to emails from WordPress backend
> * Limit registration to invited users


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/upme-invitation-codes` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the docs at http://profileplugin.com/invitation-codes-addon/ to use invitation codes.
